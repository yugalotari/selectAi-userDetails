
## Overview
This is a simple Angular application built as part of a frontend developer practical assignment.  
It demonstrates core Angular concepts such as routing, API integration, directives, dynamic components, and popup/modal interaction.

---

## Features
- **Routing:**  
  `/users` → Displays a list of users  
  `/about` → Simple About page for navigation demo

- **API Integration:**  
  Fetches user data from [JSONPlaceholder](https://jsonplaceholder.typicode.com/users)

- **Dynamic Component Loading:**  
  Components like `UserCard` and `UserDetails` are dynamically loaded.

- **Popup / Modal:**  
  Shows user details in a popup using Angular Material Dialog.

- **Directives:**  
  Custom `HighlightDirective` used for styling active users.

- **Bonus:**  
  - Search users by name  
  - Sort by name, username, or email  
  - Responsive design for better mobile experience

---

## Tech Stack
- **Framework:** Angular 17+  
- **Language:** TypeScript  
- **UI Library:** Angular Material  
- **Styling:** SCSS  
- **API:** JSONPlaceholder (Fake REST API)
